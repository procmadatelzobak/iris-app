ROOT Nadzor – vývojářské a krizové nástroje

ROOT rozhraní je elitní dashboard s pěti panely. Umožňuje ladit AI, manipulovat časem a spouštět nouzové procedury.

1) Dashboard

  • Systémový status: Zobrazuje Shift, online uživatele, teplotu a výkon. Vše se streamuje z game loopu (broadcast).
  • Výkonné protokoly: Force Shift, Global Broadcast, System Reset (NUKE) a Reload UI. Každý příkaz jde přes API s okamžitým websocket eventem.

2) CONFIG (Developer Tools)

  • Test Mode: Přepínač zobrazí tlačítka pro rychlé přihlášení na login stránce a předvyplní hesla.
  • AI Configuration: Nastavení optimizer promptu a výběr modelu pro Autopilota/Hyper (string). Uloží se do GameState a promítne do agentních terminálů.
  • API klíče: Pole pro OpenAI a OpenRouter. Gemini klíč se zatím v UI nenastavuje, ale backend jej umí načíst z env proměnné.
  • System Information: Verze appky, počet uživatelů, typ databáze.

3) Economy

  • Globální stimuly: +100/+1000 všem uživatelům. Tlačítko „Taxation“ odečte 100 kreditů (danění).
  • Reset kreditů: Vrátí všechny zůstatky na default, odblokuje Purgatory.
  • Individuální editor: Ruční změna bilance a status levelu konkrétního hráče.

4) Chronos (časová manipulace)

  • Jump to Shift: Nastaví globální offset na konkrétní číslo (0–7). Okamžitě přemapuje agenty.
  • Temperature Override: Posuvník 0–200 % navýší/poníží ochlazování a umožní rychlé spuštění glitch efektů.

5) Panopticon

  • Grid všech relací: 8×8 pohled na uživatelské a agentní zprávy bez filtrování viditelnosti.
  • Status ikony: Ikony pro lock, úkoly a overload, aby ROOT viděl, kde zasáhnout.
