Jazykové režimy - Crazy Czech

Systém IRIS podporuje více jazykových verzí. Jednou z nich je Crazy Czech - zábavná/šílená varianta češtiny určená pro zvýšení LARP atmosféry.

Co je Crazy Czech?

Crazy Czech je speciální lokalizace, která nahrazuje všechny texty v rozhraní vtipnými a absurdními alternativami. Např.:

Jak aktivovat Crazy Czech?

Změnu jazyka provádí ROOT uživatel přes svůj dashboard:

  1. Přihlaste se jako root
  2. V ROOT Dashboardu najděte sekci Nastavení jazyka (Language Settings)
  3. Vyberte crazy z nabídky jazyků
  4. Klikněte na Uložit
  5. Změna se projeví okamžitě pro všechny uživatele (po obnovení stránky)

Kdy použít Crazy Czech?

  • Pro navození atmosféry: Když chcete, aby systém působil více surrealisticky a zábavně
  • Pro eskalaci chaosu: V pozdějších fázích hry, kdy systém 'přestává fungovat'
  • Pro odlehčení: Když hráči potřebují uvolnit napětí

Důležité poznámky

  • Změna jazyka je globální - ovlivní všechny uživatele, agenty i správce
  • Vlastní názvy tlačítek (nastavené správci) zůstávají zachovány i po změně jazyka
  • Pro návrat k normální češtině vyberte czech v nastavení
