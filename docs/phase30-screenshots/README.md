# Phase 30 Screenshots - IRIS Application

This directory contains screenshots from the Phase 30 development stage of the IRIS application.

## Screenshot Categories

### Login Screen
- `00_login_screen.png` - Main login page

### User Terminals (user1-6)
- `10_user_user1_terminal.png` through `15_user_user6_terminal.png`
- Each shows the user interface for respective users

### Agent Terminals (agent1-2)
- `20_agent_agent1_terminal.png` - Agent 1 terminal
- `21_agent_agent2_terminal.png` - Agent 2 terminal

### Admin Dashboard (admin1, admin2)
For each admin user:
- `*_admin_*_hub.png` - Main HUB view with 4 stations
- `*_admin_*_monitor_all.png` - UMYVADLO (Monitoring) - All view
- `*_admin_*_monitor_chats.png` - UMYVADLO - Chats tab
- `*_admin_*_monitor_logs.png` - UMYVADLO - Logs tab  
- `*_admin_*_monitor_graph.png` - UMYVADLO - Network graph tab
- `*_admin_*_controls.png` - ROZKOÅ  (Controls) station
- `*_admin_*_economy.png` - BAHNO (Economy) station
- `*_admin_*_tasks.png` - MRKEV (Tasks) station

### Root Dashboard
- `50_root_dashboard.png` - Main dashboard view
- `51_root_config.png` - CONFIG tab (Developer Mode, AI Configuration)
- `52_root_economy.png` - GLOBAL ECO tab (Economy manipulation)
- `53_root_chronos.png` - CHRONOS tab (Time manipulation)
- `54_root_panopticon.png` - PANOPTICON tab (Surveillance grid)
