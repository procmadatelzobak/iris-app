
# PŘÍRUČKA SPRÁVCE (ROOT/ADMIN)

![System Access](/docs/images/login.png)

## 1. Přehled
Jste **SPRÁVCE** systému. Máte plnou kontrolu nad ekonomikou, časem (Shift) a konfigurací.

## 2. Dashboard
Rozhraní je rozděleno do karet:
-   **DASHBOARD:** Celkový přehled, Physics Constants (Nastavení daní, výkonu), Rychlé akce (Shift, Broadcast).
-   **CONFIG:** Nastavení jazyka, AI modelů (Task/Soft/Hyper), API klíče.
-   **ECONOMY:** Globální injekce kreditů, nastavení odměn za úkoly.
-   **CHRONOS (Time):** Ovládání posunu času (Shift) a úrovně Černobylu (Teplota).
-   **PANOPTICON:** Sledování všech aktivních relací v reálném čase.

## 3. Klíčové Funkce
-   **Force Shift:** Okamžitě posune herní čas o +1.
-   **Global Broadcast:** Odešle zprávu všem připojeným uživatelům.
-   **System Reset:** Reinicializuje herní stav (zachová účty, vynuluje teplotu).
-   **Factory Reset:** Úplné smazání dat (používat s opatrností).

## 4. Konfigurace AI
V záložce **CONFIG** můžete měnit modely pro jednotlivé agenty:
-   **Task Evaluator:** Hodnotí splnění úkolů.
-   **Optimizer:** Přepisuje zprávy agentů (Soft mód).
-   **Hyper/Autopilot:** Generuje automatické odpovědi.
